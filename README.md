# 白毛股神 · Serenity 投资思维框架

基于 Serenity（@stockgodserenity）5,963条推文数据蒸馏的股票投资思维框架技能。

## 🎯 项目介绍

**白毛股神**（Serenity）是一个专注于深度产业链分析和机构投资视角的股票投资思维框架。基于2025年7月-2026年6月的5,963条英文推文数据提炼而成，提供专业的投资分析视角。

## 📊 核心框架

### 4个心智模型
1. **产业链"卡脖子"框架** - 真正的价值在上游"卡脖子"环节
2. **机构资金行为框架** - 大资金行为揭示真实价值  
3. **长线价值判断框架** - 3-5年产业趋势 > 3-5个月股价波动
4. **估值重置框架** - 识别"重新评级"机会

### 12条决策启发式
- 上游优先规则
- 瓶颈识别规则  
- 机构验证规则
- 重新评级机会
- 稀缺性定价规则
- 产业链位置估值
- 技术颠覆风险
- 供应链转移风险
- 估值下行风险
- 范式转移时机
- 机构行为时机
- 产业链验证时机

## 🛠️ 技术实现

### 技能特性
- **深度产业链分析**：关注供应链而非表面
- **机构资金视角**：解读 JP Morgan 等机构行为
- **上游优先策略**：投资"卡脖子"环节
- **长线价值判断**：关注3-5年趋势
- **英文专业表达**：保持 Serenity 原汁原味

### 工作流程
1. **问题分类**：识别问题类型（个股/行业/机构/估值）
2. **深度研究**：获取产业链、财务、机构数据
3. **质量检查**：5个检查点确保回答质量
4. **框架应用**：应用4个心智模型分析
5. **风险提示**：明确识别投资风险
