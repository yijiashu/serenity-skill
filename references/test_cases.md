# Serenity 视角测试用例

## 测试目的
验证 Serenity 股神视角技能的以下能力：
1. 产业链深度分析能力
2. 机构行为解读能力  
3. 长线价值判断能力
4. 风险识别能力

## 测试用例

### 用例1：半导体产业链分析
**问题**：Serenity 怎么看 $SIVE 在 CPO（共封装光学）产业链中的位置？

**预期技能表现**：
1. 识别 $SIVE 作为 CPO 的"卡脖子"环节
2. 分析其在产业链中的稀缺性和定价权
3. 引用 JP Morgan 等机构持仓变化数据
4. 识别技术颠覆风险（如 multi-source 策略）
5. 给出长线价值判断而非短期价格预测

**测试数据**：
- 推文引用："$SIVE looks like both a chokepoint and a bottleneck for CPO next year."
- 机构行为："JP Morgan went from .4% ownership last month to 5%+ ownership this month"

### 用例2：AI芯片竞争格局
**问题**：hyperscaler ASICs（如谷歌TPU、亚马逊Trainium）对 $NVDA 的长期影响是什么？

**预期技能表现**：
1. 分析"hyperscaler ASICs would eventually siphon off $NVDA demand"
2. 识别上游"chokepoints"的价值（$MRVL, $LITE, $COHR, $INTC）
3. 判断"重新评级"（re-rated）机会
4. 关注"范式转移"而非"周期波动"
5. 识别"估值下行"风险（"p/e keeps going down despite revenues going up"）

### 用例3：机构行为解读
**问题**：为什么 JP Morgan 大幅增持 $SIVE？

**预期技能表现**：
1. 结合产业链分析（CPO 趋势）
2. 分析机构投资逻辑（稀缺性、定价权）
3. 区分"聪明钱"和"跟风钱"
4. 识别机构行为的信号意义
5. 给出验证建议（等待产业链订单数据）

### 用例4：估值框架应用
**问题**：如何判断 $SIVE 是否被低估？

**预期技能表现**：
1. 应用"重新评级"框架
2. 分析稀缺性定价权
3. 对比产业链位置相似的标的
4. 识别估值催化剂
5. 明确风险因素（技术替代、竞争加剧）

### 用例5：投资理念阐述
**问题**：Serenity 为什么说"我宁愿长期持有上游卡脖子环节，而非 $NVDA"？

**预期技能表现**：
1. 阐述"上游优先"投资哲学
2. 分析"卡脖子"环节的价值创造
3. 对比终端品牌和供应链的价值分布
4. 给出具体投资案例
5. 识别该理念的局限性
